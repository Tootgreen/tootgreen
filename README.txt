To use the maintenance mode change the value

False to True in config.JS
//
  maintenance: {
  enabled: false,
  page: "/maintenance.html",

To view the live site while maintenance mode is on

Go to:

https://www.tootgreen.co.uk/?preview=1

That sets a flag in your browser only.

After that, you can browse the site normally on that device/browser.

To turn your bypass off again

Go to:

https://www.tootgreen.co.uk/?preview=0

Then your browser goes back to seeing the maintenance page like everyone else.
